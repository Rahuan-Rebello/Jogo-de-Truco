# Jogo-de-Truco
Trabalho acadêmico da matéria de Desenvolvimento de Software.

Uma implementação robusta e funcional do jogo de Truco, desenvolvida integralmente em Java. O projeto foi concebido como trabalho acadêmico para a disciplina de Desenvolvimento de Software, focando na aplicação prática de Programação Orientada a Objetos (POO).

O software gerencia o ciclo completo de uma partida, desde o embaralhamento e distribuição das cartas até a lógica de pontuação dinâmica e pedidos de aumento de aposta ("Truco!").

**Destaques Técnicos:**

 - Arquitetura baseada em classes bem definidas (Cartas, Jogadores, Mesa, Partida).

 - Lógica de comparação de força das cartas (Manilhas e cartas comuns).

 - Gerenciamento de turnos e fluxo de jogo síncrono.

 - Código limpo e focado na legibilidade.
